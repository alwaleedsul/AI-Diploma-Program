# Unit 3: RNNs for Sequences | الشبكات العصبية التكرارية للتسلسلات
## AIAT 122 - Deep Learning

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand Recurrent Neural Networks (RNNs) architecture
- Implement and train LSTM and GRU networks
- Apply RNNs to sequence modeling tasks
- Handle vanishing gradient problem
- Build sequence-to-sequence models

---

## Topics Covered | المواضيع المغطاة

Based on official curriculum (AIAT 122), this unit covers:

1. **Recurrent Neural Networks (RNNs)**
   - Basic RNN architecture
   - Forward and backward propagation in RNNs
   - Vanishing and exploding gradients
   - Applications of RNNs

2. **Long Short-Term Memory (LSTM)**
   - LSTM cell structure
   - Forget, input, and output gates
   - LSTM for sequence modeling
   - Bidirectional LSTMs

3. **Gated Recurrent Units (GRU)**
   - GRU architecture
   - Reset and update gates
   - GRU vs LSTM comparison
   - When to use GRU

4. **Sequence Modeling Applications**
   - Time series prediction
   - Text generation
   - Sequence classification
   - Sequence-to-sequence tasks

5. **Practical Implementation**
   - Building RNNs with TensorFlow/PyTorch
   - Training RNNs effectively
   - Handling variable-length sequences
   - Padding and masking
