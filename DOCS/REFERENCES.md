## 1. General AI and Machine Learning | الذكاء الاصطناعي وتعلم الآلة العام

### Books

1. **"Artificial Intelligence: A Modern Approach"** by Stuart Russell and Peter Norvig
   - Comprehensive AI textbook
   - Covers all major AI topics

2. **"Pattern Recognition and Machine Learning"** by Christopher Bishop
   - Advanced ML theory
   - Mathematical foundations

3. **"Hands-On Machine Learning"** by Aurélien Géron
   - Practical ML with Scikit-Learn, Keras & TensorFlow
   - Real-world examples

4. **"Deep Learning"** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
   - Deep learning fundamentals
   - Available online: deeplearningbook.org

### Online Resources

- **Coursera:** Machine Learning courses (Andrew Ng)
- **edX:** MIT Introduction to Machine Learning
- **Fast.ai:** Practical deep learning
- **Kaggle Learn:** Free ML courses
