## ✅ Prerequisites Checklist | قائمة المتطلبات الأساسية

Before starting this unit, confirm:

- [ ] Completed Unit 1: Foundations of Generative AI
- [ ] Comfortable with probability distributions and latent spaces
- [ ] Installed required libraries (`pip check` passes)
- [ ] Reviewed related topics in `COURSE_MAP.md` if needed

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand VAE architecture
- Understand variational inference
- Build encoder and decoder networks
- Generate data using VAEs
- Compare VAEs with GANs

---

## Topics Covered | المواضيع المغطاة

1. **VAE Fundamentals**
   - Autoencoder basics
   - Variational inference
   - Latent space representation
   - Reparameterization trick

2. **VAE Architecture**
   - Encoder network
   - Decoder network
   - Latent space
   - Reconstruction loss

3. **VAE Training**
   - Loss function (reconstruction + KL divergence)
   - Training process
   - Latent space properties
   - Interpolation in latent space

4. **Applications**
   - Image generation
   - Data compression
   - Anomaly detection
   - Representation learning
