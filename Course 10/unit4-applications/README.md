# Unit 4: Generative AI Applications | تطبيقات الذكاء الاصطناعي التوليدي
## AIAT 124 - Generative AI

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Apply generative AI to text generation
- Generate images using GANs and diffusion models
- Create audio and music with generative models
- Build multimodal generative applications
- Evaluate generative model outputs

---

## Topics Covered | المواضيع المغطاة

Based on official curriculum (AIAT 124), this unit covers:

1. **Text Generation Applications**
   - Language model fine-tuning
   - Creative writing with GPT
   - Code generation
   - Text summarization
   - Dialogue systems
   - Translation and paraphrasing

2. **Image Generation Applications**
   - Image synthesis with GANs
   - Style transfer
   - Image inpainting and completion
   - Super-resolution
   - Image-to-image translation
   - Diffusion models for images

3. **Audio and Music Generation**
   - Music generation
   - Speech synthesis
   - Audio style transfer
   - Voice cloning
   - Sound effect generation

4. **Multimodal Applications**
   - Text-to-image generation
   - Image-to-text generation
   - Video generation
   - Cross-modal retrieval
   - Multimodal dialogue systems

5. **Creative Applications**
   - AI art generation
   - Content creation
   - Design assistance
   - Storytelling
   - Game content generation

6. **Practical Implementation**
   - Using pre-trained models
   - Fine-tuning for specific tasks
   - Prompt engineering
   - Output quality control
   - Integration with applications
