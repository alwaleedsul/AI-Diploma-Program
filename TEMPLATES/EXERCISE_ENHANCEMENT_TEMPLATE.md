# Exercise Enhancement Template
## Template for Creating Clear, Mastery-Focused Exercises

**Instructor Only:** This is a template. Students should not use it directly.

---

## Exercise Type 1: Guided Practice (Scaffolded)

### Template Structure:

```markdown
# Exercise X: [Exercise Name] | [الاسم بالعربي]

## Learning Objectives | أهداف التعلم
By completing this exercise, you will:
- [Objective 1]
- [Objective 2]
- [Objective 3]

## Prerequisites | المتطلبات الأساسية
Before starting, make sure you:
- [ ] Understand [concept 1]
- [ ] Can use [tool 1]
- [ ] Completed [previous exercise]

## Scenario | السيناريو
[Real-world context: Why are we doing this? What problem does it solve?]

## Instructions | التعليمات

### Step 1: [Step Name]
**What to do:** [Clear instruction]
**Why:** [Explanation of why this step matters]
**Hint:** [Helpful hint if needed]

```python
# TODO 1: [Specific task]
# Hint: [Helpful hint]
# Your code here: _______________

# Expected result: [What should happen]
```

### Step 2: [Step Name]
[Same structure]

## Solution | الحل
[Complete solution with detailed comments explaining WHY, not just WHAT]

## Common Mistakes | الأخطاء الشائعة
1. **Mistake:** [What students often do wrong]
   - **Why it happens:** [Reason]
   - **How to fix:** [Solution]
   - **How to avoid:** [Prevention]

2. **Mistake:** [Another common mistake]
   [Same structure]

## Extension | التوسع
**Challenge yourself:**
- [Extension idea 1]
- [Extension idea 2]
- [Extension idea 3]

## Self-Check | التحقق الذاتي
After completing, verify:
- [ ] Code runs without errors
- [ ] Output matches expected result
- [ ] I can explain what each part does
- [ ] I could do this again without looking

## Next Steps | الخطوات التالية
After mastering this exercise:
- Move to: [Next exercise]
- Practice: [Related skill]
- Apply to: [Real-world scenario]
```

## Key Principles for All Exercises

### 1. **Clear Instructions**
- Use simple, direct language
- Break into small steps
- Provide examples

### 2. **Context & Motivation**
- Explain WHY, not just WHAT
- Connect to real-world
- Show relevance

### 3. **Progressive Difficulty**
- Start easy, build complexity
- Each step builds on previous
- Challenge at the end

### 4. **Support & Guidance**
- Hints available (but hidden)
- Solutions provided
- Common mistakes explained

### 5. **Self-Assessment**
- Clear success criteria
- Reflection questions
- Next steps guidance

### 6. **Mastery Focus**
- Not just "complete"
- Must understand WHY
- Must be able to apply
