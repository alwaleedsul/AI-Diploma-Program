# Unit: Data Visualization

## ✅ Prerequisites Checklist | قائمة المتطلبات الأساسية

Before starting this unit, confirm:

- [ ] Completed Unit 2: Data Cleaning and Preparation
- [ ] Comfortable with Pandas and basic data summaries
- [ ] Installed required libraries (`pip check` passes)
- [ ] Understand data types and distributions
- [ ] Reviewed related topics in `../../COURSE_MAP.md` if needed

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand key concepts in data visualization
- Apply techniques and methodologies
- Work with real-world datasets
- Implement solutions to practical problems

---

## Unit Duration | مدة الوحدة

**Theoretical Hours:** 6 hours  
**Practical Hours:** 13 hours  
**Total Hours:** 19 hours

**Format:** 6 theoretical + 13 practical hours

## Topics Covered | المواضيع المغطاة

1. **Matplotlib Basics**
2. **Seaborn for Statistical Visualization**
3. **Plotly for Interactive Visualizations**
4. **Customization and Styling**
5. **Dashboard Creation**

## Files Structure | هيكل الملفات
