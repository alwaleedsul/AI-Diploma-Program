## Quiz Format

Each quiz follows this structure:
- **Time Limit:** 30 minutes
- **Total Points:** 30 points
- **Format:** Multiple choice, code writing, short answer
- **Language:** Bilingual (Arabic/English)

1. Complete the unit examples and exercises first
2. Review the unit material
3. Take the quiz without referring to materials
4. Check your answers using the answer key (instructor provides this separately)
5. Review areas where you scored below 80%
