## ✅ Prerequisites Checklist | قائمة المتطلبات الأساسية

Before starting this unit, confirm:

- [ ] Completed Unit 1: Regression Algorithms
- [ ] Comfortable with linear and polynomial regression
- [ ] Understand model evaluation basics (MSE, R²)
- [ ] Installed required libraries (`pip check` passes)
- [ ] Reviewed related topics in `COURSE_MAP.md` if needed

### Learning Objectives

By the end of this unit, students will be able to:
- Implement Ridge and Lasso regression for regularization
- Use cross-validation techniques
- Evaluate regression models using multiple metrics
- Perform feature scaling and selection
- Understand overfitting and underfitting

## Topics Covered

1. **Advanced Regression Techniques**
   - Ridge Regression (L2 regularization)
   - Lasso Regression (L1 regularization)
   - Elastic Net

2. **Model Evaluation**
   - Mean Squared Error (MSE)
   - Root Mean Squared Error (RMSE)
   - Mean Absolute Error (MAE)
   - R² Score
   - Adjusted R²

3. **Cross-Validation**
   - K-Fold Cross-Validation
   - Stratified K-Fold
   - Leave-One-Out Cross-Validation

4. **Feature Engineering**
   - Feature scaling importance
   - Feature selection techniques
   - Handling multicollinearity
